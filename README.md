# Abadi Farm Yogyakarta

Website penjualan kambing kurban dan aqiqah berkualitas di Yogyakarta.

**Live site:** [https://abadifarm.store](https://abadifarm.store)

## Tech Stack

- **Framework:** Next.js 14 (Pages Router)
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI + shadcn/ui
- **Animation:** Framer Motion
- **Database:** Supabase
- **Deployment:** Vercel

## Getting Started

1. Clone the repository
2. Copy `.env.example` to `.env` and fill in the values
3. Install dependencies: `npm install`
4. Run the development server: `npm run dev`

## Slideshow Images

The slideshow uses 3 images stored in `public/images/`:
- `slide-banner.jpg` - Abadi Farm Yogyakarta banner
- `slide-tiers.jpg` - Tingkatan (Tier) Kambing chart
- `slide-goat.jpg` - Goat product photo

Replace the placeholder SVG files with actual JPG images for production.
